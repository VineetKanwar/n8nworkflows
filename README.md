Welcome! 👋

This repository contains my collection of n8n workflows built while learning and experimenting with workflow automation, AI integrations, and API orchestration.

Although these began as practice projects, each workflow represents a step in my journey of understanding how real-world automation systems are designed, connected, and optimized.

🚀 About This Repository

The goal of this repository is to:

Learn workflow automation with n8n
Explore AI-powered automations using LLMs
Integrate third-party APIs and services
Practice data processing and workflow logic
Build reusable automation templates
Document my progress as I continue learning


📂 Repository Structure

Each project folder contains:

workflow.json – Exported n8n workflow
README.md (when applicable) – Project-specific documentation
Screenshots or supporting files (if required)

📈 Current Status

This repository is actively evolving as I continue learning.

You'll notice workflows ranging from simple automation exercises to more complex multi-step AI pipelines. As my skills grow, existing workflows will be improved and new projects will be added regularly.

🔒 Credentials & Security

For security reasons, exported workflows do not contain API keys, credentials, or sensitive configuration.

If you'd like to run a workflow yourself, simply import the JSON into your own n8n instance and configure your credentials.
