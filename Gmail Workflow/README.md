Overview

This workflow automates appointment requests received via Gmail by validating the requested meeting date and sending an appropriate response.

Workflow
Monitors incoming Gmail messages.
Extracts the requested appointment date.
If the date is within the next 7 days, confirms the appointment.
If the request falls outside the 7-day window, sends a response asking the sender to contact again later.

Outcome

Reduces manual scheduling effort while ensuring appointment requests follow predefined business rules.
