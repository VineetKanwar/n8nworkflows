Overview

This workflow extracts structured information from invoices uploaded to Google Drive using AI-powered document processing.

Workflow
Detects newly uploaded invoices.
Processes the document using an LLM.
Extracts key entities such as invoice number, vendor, dates, totals, taxes, and other relevant fields.
Returns the extracted information in a structured format.
Technologies
n8n
Google Drive
OpenAI
Document Processing
Outcome

Automates invoice data extraction, reducing manual entry and improving processing efficiency.
